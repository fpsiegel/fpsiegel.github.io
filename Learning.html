    Those are the following sections:
  • Basic Math - A nice step by step guide getting you through the basic mathematical concepts
  • Introduction to computer - A self made 5 step program to get you started with the computer
  • Web development- Nice guided exercices about HTML, CSS, JavaScript, SQL, PHP, Jquery and more!

How should one proceed ?

This depends on you. What is it that you would like to achieve ? Where do you feel comfortable starting ?
If you feel like you could use a little guidance, don't hesitate to ask you peers or anyone around you.


If it was me, I'd first go and make sure my mathematic skills are up-to-date on KAHN ACADEMY or SCHOOL YOURSELF. Then I'll go through the amazing intro to coding for Grades K-5 on CODE.ORG. If I enjoyed it, I'd continue their programs, and if I'd like some change, I'd visit CODECADEMY and give a try to their teaching method which is very nice and detailled. Finally, when I feel confident, I'd start working with LEARN PYTHON or LEARN CPP and play around with all their amazing functions!

Once I am a bit confident about HMTL and CSS, I'd also create my own repository on GITHUB , it is free and allow to get a website of your own, where you can showcase your work, it is nice to build it and improve it over time, and if people are interested in what you do, they can even help you through github branch/pull system!
